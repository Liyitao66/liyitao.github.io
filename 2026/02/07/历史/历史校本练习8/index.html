历史校本练习8参考答案

一、选择题答案

  1. B
    （1935年华北事变，中日民族矛盾上升为主要矛盾，“八一宣言”顺应民族革命需求；抗日民族统一战线建立（A）是1937年，全国团结抗战（C）是1937年，敌后抗日根据地开辟（D）是1937年后）

  2. D
    （中华民国二十五年为1936年，事件为西安事变，解决表明全国团结抗战局面初步形成；全面抗战路线（A）是1937年洛川会议，抗日民族统一战线正式形成（B）是1937年，局部抗战开始（C）是1931年）

  3. D
    （国共合作抗战宣言发表，标志国共合作抗日；洛川会议（A）是1937年，遵义会议（B）是1935年，蒋介石庐山讲话（C）是1937年）

  4. C
    （日本推行奴化教育，毒害青少年；促进文化交流（A）、解决子女教育（B）、破坏教育体系（D）非主旨）

  5. A
    （民族工业内迁，支持敌后战场；美国支援（B）、东西经济差距（C）、全面爆发（D）均错误）

  6. C
    （百团大战是抗战时期中共发动的最大规模战役；徐州会战（A）、太原会战（B）、淞沪会战（D）均错误）

  7. C
    （参议会人员构成反映建设抗日民主政权；敌后游击战争（A）、减租减息（B）、抗战宣传（D）均错误）

  8. B
    （减租减息政策团结地主阶级抗日；缓和与国民党矛盾（A）、国际社会援助（C）、联合资产阶级（D）均错误）

  9. C
    （中国战场是世界反法西斯主战场；美国决定性作用（A）、印度主战场（B）、日本战线过长（D）均错误）

  10. A
    （中共七大主题是打败日本侵略者，建立新民主主义新中国；全面建设小康（B）、从农业国变工业国（C）、改革开放（D）均错误）

  11. B
    （漫画反映国共谈判，国共两党北平和平谈判破裂（B）；全国人民争取民主（A）、国民党独裁（C）、解放战争胜利（D）均错误）

  12. C
    （《中国土地法大纲》废除封建性及半封建性剥削的土地制度；推动国统区土地改革（A）、农民革命（B）、解放军战略进攻（D）均错误）

  13. A
    （辽沈战役胜利；国民党重点进攻（B）、挺进大别山（C）、淮海战役（D）均错误）

  14. C
    （漫画反映解放军战略反攻，为渡江作战奠定基础（C）；打破日本灭亡中国计划（A）、揭开反攻序幕（B）、增强抗战必胜信心（D）均错误）

  15. D
    （根据地建设得到广大人民群众支持；美国援助（A）、苏联支持（B）、高超战略战术（C）均错误）

二、非选择题答案

16.

(1) 十四年抗日战争开始的时间及标志性事件：
1931年九一八事变。

(2) 抗日战争胜利的历史意义：

  • 是中国近代以来反抗外敌入侵第一次取得完全胜利的民族解放战争；
  • 促进了中华民族的觉醒，为中国共产党带领中国人民实现彻底的民族独立和人民解放奠定了重要基础；
  • 对世界反法西斯战争的胜利、维护世界和平作出了巨大贡献；
  • 中国国际地位得到提高。

17.

(1) 20世纪30年代抗日救亡运动的特点：

  • 参与阶层广泛，包括工人、农民、学生、知识分子、民族资产阶级等；
  • 斗争形式多样，有罢工、请愿、游行、抵制日货等；
  • 以抗日救亡为核心目标；
  • 在中国共产党领导下或影响下开展；
  • 具有广泛的群众基础和爱国性质。

(2) 抗日战争成为“中国复兴枢纽”的原因：

  • 抗日战争促进了中华民族的觉醒和团结，增强了民族凝聚力；
  • 中国共产党在抗战中发挥了中流砥柱作用，扩大了影响力；
  • 抗战胜利提高了中国的国际地位；
  • 抗战期间民族工业内迁，促进了西南、西北地区经济发展；
  • 抗战胜利为新民主主义革命胜利奠定了基础，推动了中国社会进步。
